My Experience

Size Matters

This game was originaly created at a 48 hour game jam for MSU's game developement club Spartasoft, but has since been improved. My goal for this game was multiplayer. Due to time constraints I decided to use Photon asset for Unity3D. Although I was barely able to put together a playable game by the end of the game jam, I'm pretty happy with how it came out after a little bit of polish.

Socket Mech

For Spartahack 2017 (36 hour hackathon), I aimed to created a server. Then I soon realized I didnt' have enough time to create one, so half way through my co-worker and I scrapped the server and started making a VR game. Due to the short amount of time the code is pretty messy, which led to some bugs. For the most part, it was playable and fairly fun. We also took advantage of X-Keys, which are different size and shaped macro buttons for this project. So the layout of the buttons inside the mech suit is the same layout that was actually infront of the player.

PEG

PEG was one of my favorite games I've made so far. It was for a 48 hour Ludum Dare game jam. For this game jam, we could only work for about 20 hours, so we aimed for something simpler. The goal is to match your shape to fit inside the falling shapes. We finished early so we were able to fix some bugs, and even add a leaderboard.

Balance

For the 48 hour Ludum Dare game jam, I wanted to play around with particle effects and lighting. With that in mind, I went forward with my first split-screen co-op game. Although, due to time constraints, the map is a little underwelming, I ended up learning a lot about the potential of particle effects

Temple Of The Mask

Global Game Jam 2016 was my first experience working with a team, and one of my first big Unity projects. It is riddled with bugs and the code is a bit shotty, but one of my biggest motivaters to pursue game developement

Blorp

Blorp is a Spartasoft (MSU game developement club) Prespec(4 week team game developement). This was a great team effort, where everyone really had their hearts in it. On this game I tried to learn bettery design from the more experienced teamates.
